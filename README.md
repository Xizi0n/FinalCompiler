# FinalCompiler

A tudástár alkalmazás távoli fordító szervere.

## Előfeltételek:
- [Docker](https://www.docker.com/)
- [NodeJs](https://nodejs.org/en/)
- Linux

## Beüzemelés

1. `git clone https://github.com/Xizi0n/FinalCompiler.git`

2. `cd FinalCompiler`

3. `npm install`

4. `./init.sh`

5. `node index.js`
